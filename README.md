# Project Overview
- Built a visually interactive sales dashboard to analyze, track, and present e-commerce business performance.
- Enables quick insights into sales, profit, quantity, and top products.

---

# Dataset
- **Sample Superstore** dataset (50,000+ rows, multiple regions & products)

---

# Tools & Technologies
- Power BI Desktop
- Microsoft Excel (for initial data cleaning)

---

# Key Features
- **KPI Cards:** Real-time Total Sales, Total Quantity, Total Profit
- **Slicer Filters:** Instant filtering by Category and Region
- **Top Products Table:** Product-level Sales, Quantity, Profit breakdown
- **Product Search:** Find any product instantly with filter/search box
- **Modern Theme:** Custom professional color theme and clean layout
- **Actionable Insights:** Spot underperforming categories or products for strategic recommendations

---

# Insights & Business Value
- Quickly identify sales trends and best-selling products by segment & geography
- Drill-down filtering for granular business questions
- Reduced reporting time by automated interactive dashboard


---

# How to Use
1. Open `.pbix` file in Power BI Desktop or view on Power BI Service via the live link
2. Use side panel filters to analyze by category or region
3. Search for products using the right filter pane
4. Use the main KPIs and table to answer key business queries
